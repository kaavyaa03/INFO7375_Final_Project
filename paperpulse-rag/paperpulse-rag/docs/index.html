PaperPulse RAG

Prompt Engineering + Retrieval-Augmented Generation (RAG) System

What this project does

PaperPulse RAG is a generative AI system that answers questions and generates summaries by grounding responses strictly in retrieved documents. It demonstrates two core components: Prompt Engineering and RAG.

Key features

  • Knowledge base ingestion from local documents (data/kb_samples)
  • Chunking with overlap to preserve context
  • Vector storage + retrieval using FAISS (data/vectorstore)
  • Prompt-engineered outputs with citations and safe fallback behavior
  • Metrics included in responses (e.g., retrieved chunk count)
  • Interactive demo via Streamlit UI + FastAPI backend

How to run (local demo)

  1. Install dependencies: pip install -r requirements.txt
  2. Start backend: python -m uvicorn app.api:app --reload --port 8000
  3. Start UI: streamlit run app/ui_streamlit.py
  4. Click Build Knowledge Base and ask questions in the UI.
